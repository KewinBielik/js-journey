## Step 2 — Concepts (read, then do)

A **variable** is a labeled box that stores a value.

```js
let age = 30;        // a box named "age" holding the number 30
const name = "Kewin"; // a box named "name" holding text (a "string")
```

- `let` = a box whose contents you can change later.
- `const` = a box whose contents you CANNOT change (use this by default — it's safer).

The three data types you'll meet today:
- **String** — text, always in quotes: `"hello"`, `'JavaScript'`
- **Number** — `7`, `3.14`, `-100`
- **Boolean** — only two values: `true` or `false`

`console.log(...)` prints whatever you put inside the parentheses to the Console.
